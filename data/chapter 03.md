# **UNIT 3 – CHIA ĐỘNG TỪ THÌ HIỆN TẠI (PRESENT TENSE)**

*(Giáo trình tiếng Việt chi tiết + giải thích quy tắc chính tả đi kèm)*

***

# I. **Mở đầu – Vai trò của động từ trong câu**

Động từ trong tiếng Hà Lan diễn tả:

*   một hành động: **Erik werkt** (Erik làm việc)
*   một trạng thái: **Hij is advocaat** (Anh ấy là luật sư)
*   sự thay đổi: **Hij wordt boos** (Anh ấy trở nên tức giận)
*   sự di chuyển: **Ze gaan naar school** (Họ đi học)

Động từ **phải hòa hợp với chủ ngữ** (subject–verb agreement). 

***

# II. **Cấu trúc cơ bản của động từ Hà Lan**

Hầu hết động từ có dạng:

### **Infinitive (nguyên mẫu)**: **werken** (to work)

→ stem (gốc): **werk**  
→ thêm đuôi để chia theo ngôi.

***

# III. **Bảng chia động từ thường – Động từ *werken* (làm việc)**

| Đại từ          | Hình thức             | Ghi chú                                         |
| --------------- | --------------------- | ----------------------------------------------- |
| **ik**          | **werk**              | stem = werk                                     |
| **jij/je**      | **werkt** / *werk je* | nếu *jij/je* ĐỨNG SAU động từ → **không có –t** |
| **u**           | **werkt**             | luôn có –t                                      |
| **hij/zij/het** | **werkt**             |                                                 |
| **wij/we**      | **werken**            | luôn giống nguyên mẫu                           |
| **jullie**      | **werken**            |                                                 |
| **zij/ze**      | **werken**            |                                                 |

 

***

# IV. **Quy tắc đặc biệt về –t với *jij / je***

Khi **jij/je đứng SAU động từ**, dạng **không có –t**:

*   **Werk jij in een restaurant?**
*   **Maak je nu je huiswerk?**

 

***

# V. **Hai động từ bất quy tắc quan trọng nhất: *hebben* & *zijn***

## **1. Động từ *hebben* – “có”**

| Đại từ         | Chia               |
| -------------- | ------------------ |
| ik             | **heb**            |
| jij / je       | **hebt** (heb je?) |
| u              | **hebt / heeft**   |
| hij/zij/het    | **heeft**          |
| wij/jullie/zij | **hebben**         |

## **2. Động từ *zijn* – “là / thì / ở”**

| Đại từ         | Chia               |
| -------------- | ------------------ |
| ik             | **ben**            |
| jij/je         | **bent** (ben je?) |
| u              | **bent**           |
| hij/zij/het    | **is**             |
| wij/jullie/zij | **zijn**           |

 

***

# VI. **Động từ có gốc kết thúc bằng –t hoặc –d**

## **1. Gốc kết thúc bằng –t**

→ **KHÔNG thêm –t** nữa cho jij/hij.

Ví dụ: **zetten** (đặt)

*   ik **zet**
*   jij **zet**
*   hij **zet**

## **2. Gốc kết thúc bằng –d**

→ vẫn thêm –t ở jij/hij.

Ví dụ: **raden** (đoán)

*   ik **raad**
*   jij **raadt**
*   hij **raadt**

Trường hợp đảo ngược *jij* đứng sau động từ → có thể rơi mất –d:

*   **raad je?**

 

***

# VII. **Nhóm động từ rất đặc biệt – gốc chỉ còn một phụ âm**

Một số động từ chỉ có **1 âm tiết** trong gốc và mất –en:

| Infinitive           | ik  | jij/hij | wij  |
| -------------------- | --- | ------- | ---- |
| **gaan** (đi)        | ga  | gaat    | gaan |
| **zien** (nhìn thấy) | zie | ziet    | zien |
| **doen** (làm)       | doe | doet    | doen |

 

Lưu ý: **staan** (đứng) và **slaan** (đánh) chia giống *gaan*.

***

# VIII. **RULE SIÊU QUAN TRỌNG: ẢNH HƯỞNG TỪ CHÍNH TẢ (Unit 1 → Unit 3)**

Các quy tắc chính tả quyết định dạng chia động từ.

## 1. **Nguyên âm dài ở âm tiết đóng → viết đôi**

Ví dụ hai động từ tương phản:

### **spelen** (chơi)

*   âm tiết mở **spe–len** → nguyên âm dài → khi chia:
    *   ik **speel**
    *   wij **spelen**

### **spellen** (đánh vần)

*   âm tiết đóng **spel–len** → nguyên âm ngắn → khi chia:
    *   ik **spel**
    *   wij **spellen**

 

## 2. **Không tồn tại hai phụ âm giống nhau cuối âm tiết**

→ khi chia số ít, phụ âm kép **giảm xuống một**:

*   **zitten → ik zit**, nhưng **wij zitten**

## 3. **Biến âm f→v, s→z** khi nằm giữa nguyên âm

→ giống trong số nhiều danh từ.

Ví dụ:

| Infinitive       | ik   | wij    |
| ---------------- | ---- | ------ |
| **geven**        | geef | geven  |
| **wijzen** (chỉ) | wijs | wijzen |

# X. **Cách dùng động từ trong câu hỏi**

## 1. **Yes/No questions**

Động từ đứng đầu, theo sau là chủ ngữ.

*   **Werk jij hier?**
*   **Ga je mee?**

## 2. **Wh- questions**

Từ để hỏi + động từ + chủ ngữ

*   **Wat doe je?**
*   **Waar woon je?**

# XII. **Kết luận – Gói nhớ nhanh**

| Quy tắc                                      | Ví dụ                 |
| -------------------------------------------- | --------------------- |
| ik = stem                                    | ik werk / ik loop     |
| jij/hij = stem + **t**                       | jij werkt / hij loopt |
| khi jij đứng sau động từ → **không thêm –t** | Werk je?              |
| wij/jullie/zij = infinitive                  | wij werken            |
| gốc có –t → không thêm –t                    | jij zet               |
| gốc có –d → thêm –t                          | jij raadt             |
| đặc biệt: gaan/zien/doen                     | ik ga, jij gaat       |
| chính tả gốc quyết định chia                 | speel vs. spel        |
